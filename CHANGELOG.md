## 1.0.1

* Initial release.
* Added `check` command for full codebase analysis (Lints, Unused Packages, Complexity, Assets).
* Added `report` command to generate Markdown and JSON reports.
* Added `score` command to generate a dynamic 100-point Health Score.
* Added `fix` command to auto-fix unused packages, unused assets, and dart lints automatically.
* Added `undo` command with a built-in Backup System.
* Added support for `flutter_doctor_pro.yaml` custom configuration.
